# 🎮 1vs1 Online Dama

Gerçek zamanlı, animasyonlu 1vs1 online dama oyunu. Socket.io ile çalışır.

## 🚀 Canlı Demo

[https://online-dama.onrender.com](https://online-dama.onrender.com)

## ✨ Özellikler

- ✅ 1vs1 gerçek zamanlı oyun
- ✅ Animasyonlu taş hareketleri
- ✅ Çoklu yeme desteği (+ sayaçları)
- ✅ Yenen taşlar koleksiyonu
- ✅ Dama (king) dönüşümü
- ✅ Responsive tasarım (mobil/tablet/desktop)
- ✅ Oda sistemi (6 haneli kod)
- ✅ Sayfa yenilemeye dayanıklı

## 🛠️ Teknolojiler

- Node.js
- Express
- Socket.io
- HTML5/CSS3
- JavaScript (Vanilla)

## 📦 Kurulum

```bash
# Repoyu klonla
git clone https://github.com/yourusername/online-dama.git
cd online-dama

# Bağımlılıkları kur
npm install

# Geliştirme modunda çalıştır
npm run dev

# Produksiyon modunda çalıştır
npm start
